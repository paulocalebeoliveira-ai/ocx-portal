📋 Template — Credenciais de acesso (enviado ao criar usuário)
Variáveis disponíveis: {nome}, {login}, {senha}
Como obter a chave Brevo
  1. Acesse brevo.com e crie uma conta gratuita
  2. Vá em SMTP & API → Chaves de API
  3. Clique em "Criar nova chave de API" → dê um nome → copie a chave
  4. Cole no campo acima e clique em Salvar configurações
  5. Configure o e-mail remetente (verifique o domínio no Brevo se necessário)
